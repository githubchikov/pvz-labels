# pvz-labels
Программа для автоматизация приемки товаров в ПВЗ - автоматическая печать номеров ячеек на термопринтере

## Интерфейс

### Статус
Показывает текущий статус программы

### Область распознавания
Область, в которой должно происходить распознавание номера ячейки. Во время распознавания текста на экране доступен предпросмотр выбранной области

### Принтер
Программа работает преимущественно с принтерами этикеток. При выборе обычного принтера печать может происходить криво, либо с ошибками. Для выбранного принтера доступна кнопка открытия его свойств и отправка пробной печати - этикетка с номером "123"

### Размер этикетки
Ширины и высота этикетки, используемая в принтере. Расположение и размер текста во время печати подстраиваются под размер этикетки. Учитывайте, что слишком длинные номера ячеек могут не помещаться на этикетке. 

### Параметры печати

#### Разрешенные символы
Символы, которые будут распознаваться на экране. Данный фильтр ограничивает область распознаваемых символов во избежании печати лишнего контента.
Указываются без пробела, символы по умолчанию: 0123456789-

#### Минимальное количество символов
Количество символов, от которых будет начинаться печать. Рекомендуется указывать минимум 2

#### Интервал опроса экрана, мс
Интервал обновления распознавания экрана в миллисекундах. Чем меньше значение, тем чаще выполняется распознавание и тем выше нагрузка на процессор. Рекомендуемое значение — 300 мс

### История печати
Последние распознанные и распечатанные номера. Указывается дата и время распознания и сам номер
